CSV Format

CSV2Cal supports the following CSV columns:

  • Subject - required
  • Start Date - required
  • Start Time
  • End Date
  • End Time
  • Location
  • Description
The default timezone is UTC+11(Australia/Sydney)

Example CSV format

Subject,Start Date,Start Time,End Date,End Time,Location,Description
[Div 5] - RTTCD5A vs. RTTCD5B @RTTC1 on 21st Feb '24,21/Feb/24,18:45,,20:45,RTTC1,
[Div 5] - RTTCD5B vs. PCYC5F @RTTC1 on 1st Mar '24,1/Mar/24,18:45,,20:45,RTTC1,
